# CSVエクスポート

スキャン・確認が完了したら、仕訳データを CSV でダウンロードして会計ソフトに取り込みます。

## ダウンロード手順

結果画面の最上部にある**「全件 CSV をダウンロード (N 件)」**ボタンをクリックします。

- `仕訳データ.csv` がダウンロードされます
- 編集済みの内容（取引先・金額・勘定科目）が反映されます
- 確認不要の仕訳もすべて含まれます

## CSVの列構成

| 列 | 説明 |
|----|------|
| 取引日 | YYYY-MM-DD |
| 取引先 | AI 読取値またはユーザー編集値 |
| 税込金額 | 修正した場合は税抜・消費税が空欄に |
| 税抜金額 | AI 計算値 |
| 消費税額 | AI 計算値 |
| 勘定科目 | AI 推奨またはユーザー選択 |
| 補助科目 | （将来対応） |
| 課税区分 | 課税仕入れ / 軽減税率 / 非課税 等 |
| 税率(%) | 10 / 8 / 0 |
| 推奨ファイル名 | 電帳法準拠ファイル名 |
| 備考 | ユーザー入力のメモ |

## 対応する会計ソフト

ダウンロードした CSV は、以下の会計ソフトにそのまま取り込めます。

- **freee** — 振替伝票形式
- **マネーフォワード** — 仕訳帳形式
- **弥生会計** — 汎用 CSV 形式
- **汎用 CSV** — どのソフトでも取込可能な標準形式

> Phase 1 では汎用 CSV のみの対応です。各ソフト専用形式は Phase 1.5 以降で対応予定です。

## 個別画像のダウンロード

各結果カードのファイル名右にある 💾 ボタンで、電帳法ファイル名が付与された個別画像をダウンロードできます。

## 索引 CSV（電帳法対応）

CSV エクスポート時、電帳法の検索性要件を満たす索引データが同列に含まれます。税務調査時に「日付・金額・取引先」で検索できる状態を保証します。
