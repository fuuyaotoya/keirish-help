# 料金プラン — Free と Advice の違い

ケイリッシュは「まず無料で試す → 必要になったら課金」のシンプルな 2 段階構成です。

## プラン比較

| 機能 | Free | Advice |
|---|---|---|
| 月額 | **¥0** | **¥1,980 / 月** |
| 仕訳件数 | 30 件 / 月 | **無制限** |
| AI アドバイス | 3 回 / 月 | **無制限** |
| 領収書スキャン | ✓ | ✓ |
| 電帳法準拠保存 | ✓ | ✓ |
| 会計ソフト形式 CSV | ✓ | ✓ |
| Excel エクスポート | — | ✓ |
| Google Sheets 連携 | — | ✓ |
| 税理士向け共有 | — | ✓ |

## 先着 100 名特典

ローンチ後、先着 **100 名** には **¥1,980 永年特典**を適用。
通常料金改定後も値上げの影響を受けません。

## 月の上限に達したら

Free プランで月 30 件 (仕訳) または 3 回 (AI アドバイス) の上限に達すると、
画面上に「アップグレード」モーダルが表示されます。

Advice にアップグレードすると **その場で制限が解除** されます。月初に自動でカウンタはリセットされます。

## 解約

「設定」→「請求」→「Stripe カスタマーポータル」から、ワンクリックで解約できます。
解約後も期末まで Advice 機能を利用可能です。

## よくある質問

**Q. 無料プランでも電帳法対応の保存はできますか？**
A. はい。Free プランでも索引 CSV 同梱・SHA-256 改ざん防止は標準機能です。

**Q. 1 回課金して、すぐ解約してもいい？**
A. もちろん可能です。ただし「先着 100 名特典」は連続契約時のみ有効です。

**Q. 領収書の保存先は？**
A. Phase 1 ではエクスポートのみ。データはダウンロード後にサーバー側で破棄します。
Phase 2 (ALL プラン) で長期保存版を提供予定です。
