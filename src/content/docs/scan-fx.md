# 外貨の領収書

ケイリッシュは主要10通貨の外貨建領収書に対応しています。USD や EUR で記載された領収書をアップロードすると、自動的に通貨を検知して換算を行います。

## 対応通貨

USD / EUR / CNY / KRW / GBP / AUD / HKD / SGD / CAD / CHF

## 外貨検知時のオンボーディング

外貨の領収書を初めてスキャンすると、換算方法の選択モーダルが表示されます。

> 「🌏 USD の領収書を検知しました — 円換算をどうしますか?」

### 3つの換算モード

| モード | 説明 | 対象 |
|--------|------|------|
| **簡単**（推奨） | その日の TTM レートで自動換算。ざっくり経営者向け。9割の方はこれ | Phase 1.0 対応 |
| **正確** | カード明細を待って実レートで精算。仮計上は TTM、後日カード明細で為替差損益も自動仕訳 | Phase 1.5 予定 |
| **月固定** | 月初の TTM で月内固定。法人税法の継続適用が楽 | Phase 1.5 予定 |

> Phase 1.0 では「簡単」モードのみ利用可能です。「正確」「月固定」を選択した場合も、当面は「簡単」と同じ扱いになります。

### 「全件・全通貨に適用」チェック

「この方法を全件・全通貨に適用（年度内固定）」にチェックを入れると、以降の外貨スキャンでも同じ換算方法が自動適用されます。

> ⚠ 法人税法上、換算方法は年度内継続適用が必要です。一度選択したら年度内は同じ方法を使い続けてください。

## 換算結果の表示

外貨領収書の結果カードには、換算行が追加表示されます。

```
USD 50.00 → ¥7,461 [TTM]
```

クリックすると詳細が展開され、以下の情報が確認できます。

- 換算計算式
- レート種別（TTM: みなし仲値）
- レート取得元（ExchangeRate-API / ECB）
- 取得日時

> **注意**: 自動取得レートは市場参照レートであり、銀行公示 TTM（電信売買相場の仲値）ではありません。税務上の正確性が必要な場合は、銀行の公示 TTM を手動で入力してください。

## 外貨のファイル名規則

円換算額が確定している場合:

```
YYYYMMDD_円換算額JPY_通貨コード外貨額.拡張子
```

例: `20260515_15000JPY_USD100.00_AmazonWebServices.pdf`

円換算額が未確定の場合:

```
YYYYMMDD_通貨コード外貨額_取引先.拡張子
```
