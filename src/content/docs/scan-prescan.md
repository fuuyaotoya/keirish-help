# スキャン前の確認

ファイルをアップロードすると、「仕訳前の確認」パネルが表示されます。ここでスキャンの前提条件を設定します。

## ① 支払い手段を選択

どの支払い方法で購入したかを選択します。複数選択可能です。

| 選択肢 | 説明 |
|--------|------|
| 現金 | 現金で支払った領収書 |
| カード | クレジットカード決済 |
| 普通預金 | 銀行振込 |
| 役員借入金 | 役員から立て替えてもらった場合 |
| その他 | 上記以外 |

> 💡 選択しなくてもスキャンは可能です。AI が領収書の内容から推測します。

## ② 期間を入力（任意）

「例: 2026年5月」のように期間を入力すると、AI が取引日の推定に活用します。特に月をまたぐまとめスキャン時に便利です。

## スキャンを実行

「仕訳を開始 (約N枚・約1分)」ボタンをクリックすると、AI 解析が始まります。

### 処理中の表示

スキャン中は青いプログレスカードが表示され、以下の3段階で進行します。

1. **PDF を分解中...** — PDF の場合、ページごとに分割
2. **AI 解析中...** — Gemini Vision が各画像を解析
3. **領収書を切り出し中...** — PDFから領収書部分を抽出

処理中は他の操作はできません。通常は数秒〜数十秒で完了します。

### エラーが出た場合

- 「領収書を検出できませんでした」→ 画像に領収書が含まれていない可能性があります。別の画像でお試しください。
- クォータ超過 → [料金プラン](/pricing/) の上限に達しています。アップグレードが必要です。
