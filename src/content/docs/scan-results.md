# AI解析結果の見方

スキャンが完了すると、結果が**3つのレーン（グループ）**に分類されて表示されます。

## 「全件 CSV をダウンロード」ボタン

結果画面の最上部に、一番大きなボタンが配置されています。**確認不要で即ダウンロード可能**です。

> 💡 確認が不要な場合は、このボタンだけ押して終了できます。これが「3クリック経理」の3つ目のクリックです。

## 3つのレーン

### 要確認（⚠ マーク付き）

AI が自信を持てなかった仕訳です。内容を確認して、必要に応じて修正してください。

「要確認」になる理由は以下のとおりです。

| 理由バッジ | 意味 |
|-----------|------|
| 手書き | 手書き文字が含まれており、OCR精度が下がる可能性 |
| 検算エラー | 税込×税率から逆算した税抜額と合わない |
| 高額 | 過去の同種取引より大幅に高い金額 |
| ブラー検出 | 画像がぶれている |
| OCR自信度 | 文字認識の確信度が低い |
| 勘定科目自信度 | 勘定科目の推定に自信がない |

各カードには理由バッジが表示されるので、**なぜ確認が必要なのか**がひと目でわかります。

### 自動処理済

AI が高確度で仕訳を確定したものです。折りたたみ式になっており、クリックで展開して内容を確認できます。

修正の必要がない場合はそのままで問題ありません。

### エラー

領収書として検出できなかった画像です。以下のようなケースが該当します。

- 領収書以外の画像がアップロードされた
- 画像が極端に暗い・ぶれている
- 文字が全く読み取れない

## Wowバッジ（学習機能）

過去に同じ取引先の領収書をスキャンしたことがある場合、「過去 N 回『○○』にされていたので、今回も同じ科目にしました」というバッジが表示されます。

これはブラウザ内の学習履歴（localStorage）に基づく推測です。サーバーには保存されません。

## 画像ハイライト

領収書の画像上で、AI が読み取った箇所が色分けされてハイライト表示されます。

| 色 | 対応フィールド |
|----|---------------|
| 🔵 青 | 支払先 |
| 🟢 緑 | 金額 |
| 🟠 オレンジ | 取引日 |

入力欄にマウスオーバーすると対応するハイライトが光る**双方向連動**機能があります。
